# Card.lol — Profile Card Builder

A guns.lol-style profile card builder. Create an account, customize your card, share your link.

## Run

```bash
npm install
node server.js
```

Then open **http://localhost:3000**

## Features

- **Account** — Sign up, log in, logout
- **Settings** — Left panel with all options, right panel with live preview
- **Profile** — Display name, description/bio, profile picture
- **Second PFP** — Optional second image that appears on hover
- **Links** — Add multiple links with titles
- **Hover effects** — Tilt X/Y (degrees), scale, glow, animation duration
- **Card styling** — Background color/gradient/image, blur, accent color, border, radius, shadow
- **Typography** — Font choice (Syne, DM Sans, Inter, Space Mono, Playfair Display)
- **PFP shape** — Circle, rounded square, or square

## Your profile

After creating an account, your public profile lives at:

**http://localhost:3000/c/yourusername**
